# AIJudge — AI Settles Your Arguments

Describe any argument or disagreement — AI delivers an impartial verdict with reasoning, roasts both sides, and generates a shareable court ruling card. Share it with your group chat.

## Features

- **Impartial verdicts** — AI analyzes both sides and picks a winner (or calls it complicated)
- **Party breakdown** — strength scores, valid points, and weaknesses for each side
- **Legal-style reasoning** — detailed court reasoning that references both arguments
- **Humorous roasts** — dry humor observations about the argument
- **Shareable PNG cards** — download or copy a screenshot-worthy ruling card
- **Past rulings history** — browse your previous cases from localStorage
- **Appeal option** — disagree? Send the same case back for a re-ruling

## How It Works

1. **Describe the argument** — include both sides for a fair ruling
2. **AI judges who's right** — delivers a verdict with full reasoning
3. **Share the ruling card** — download the PNG and send it to your group chat

## Tech Stack

- Vanilla HTML, CSS, JavaScript (no frameworks)
- Cloudflare Worker proxy to OpenRouter (Claude Sonnet)
- html2canvas for shareable ruling images
- Zero signup, zero paywall
